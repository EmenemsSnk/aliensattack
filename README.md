# Aliens Attack

A classic 2D space shooter game built in Java using Swing.

## Description
Take control of a spaceship and defend against waves of invading aliens! Move your ship and fire missiles to destroy the alien fleet before they can reach you.

## Features
- Player-controlled spaceship
- Moving alien enemies
- Missile firing mechanics
- Collision detection

## How to Play
- **Arrow Keys**: Move the spaceship (Up, Down, Left, Right)
- **Spacebar**: Fire missiles

## Requirements
- Java 8 or higher
- Maven

## How to Build and Run
```bash
mvn clean compile
mvn exec:java -Dexec.mainClass="com.emenems.games.aliens.Main"
```
